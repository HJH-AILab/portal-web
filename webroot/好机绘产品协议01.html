二、账号注册及管理

2.1 注册资格

  • 您需具备完全民事权利能力和行为能力（自然人、法人或其他组织）。

  • 未成年人或限制民事行为能力人不得注册，若注册需监护人承担全部后果。

  • 您需确保非任何国家 / 地区贸易限制、制裁对象，否则可能无法注册或使用服务。

2.2 账号注册与登录

  • 完成注册流程并同意本协议后，您将获得好机绘账号及唯一 “账号 ID”（含账号名、手机号及关联信息）。

  • 账号名需符合法律法规及平台规范，不得混淆用户与我们的身份，否则可能被拒绝注册或注销。

  • 需准确提供并及时更新账号信息（如姓名、企业信息等），因信息不实导致的账号冻结、注销等后果由您承担。

  • 我们有权根据关联信息（如同一证件、手机号、设备、API 密钥等）判断多个账号是否属于同一用户，并进行统一管理。

2.3 账号使用与安全

  • 您有权使用账号登录本产品及 API 服务，但我们有权变更、暂停或取消服务（含功能调整、版本升级等），并通过公告、站内信、短信等方式通知。若不同意变更，请勿继续使用。

  • 账号仅限注册主体使用，未经我们同意，不得转让、出租、赠与或继承（API 服务账号需绑定唯一主体）。

  • 需妥善保管账号、密码及 API 密钥，确保登录后安全退出。如发现未经授权使用，需立即通知我们，我们将协助处理，但对处理前产生的后果不承担责任。

  • 我们将采取行业标准安全措施保障账号安全。

三、账号的冻结、注销及申诉

3.1 账号冻结

账号（含 API 服务权限）可能因以下原因被冻结：

  • 违反本协议、平台规则或 API 使用规范；

  • 被投诉且未提供反证；

  • 账号操作、收益或 API 调用存在异常；

  • 违反国家法律法规或监管要求；

  • 其他类似风险行为。

3.2 账号注销

  • 主动注销：您可提交注销申请，我们将按流程处理；

  • 强制注销：若冻结情形严重或应监管要求，我们将注销账号；

  • 自动注销：连续 12 个月未登录本产品（含冻结期），我们有权注销账号。

  • 注销后果：无法登录、找回数据，且不豁免注销前的行为责任。

3.3 申诉

  • 账号被冻结或注销后，您可提交申诉，需提供身份证明、账号信息及相关材料。我们有权决定是否受理。

  • 若拒绝提供材料或审核不通过，我们有权长期冻结或注销账号。

3.4 关联账号处理

若您名下多个账号存在违规行为，我们有权对其他账号一并冻结或注销。
